# Willow Spence Urban Retreat

Luxury property listing website for a 16-bedroom rental in Austin, Texas.

## Property Details

- **Name**: Willow Spence Urban Retreat
- **Address**: 1004 Spence Street, Austin, Texas 78702
- **Bedrooms**: 16
- **Capacity**: 32+ guests
- **Type**: Luxury group accommodation

## Features

- Responsive property listing page
- Interactive photo gallery with lightbox
- 62 high-quality property photos
- Modern, elegant design
- Mobile-friendly interface

## Usage

Simply open `index.html` in a web browser to view the property listing.

## Photo Gallery Controls

- **Click thumbnails**: Opens full-screen lightbox
- **Navigation arrows**: Browse through photos
- **Keyboard shortcuts**:
  - Arrow keys: Navigate photos
  - ESC: Close lightbox
- **Touch gestures**: Swipe on mobile devices
- **Click outside**: Close lightbox

## Development

This is a static HTML site with no build process required. All assets are self-contained.
